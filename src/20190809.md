# 人是怎么废掉的

毕业工作五年后依然决定去考研。当周围的同学和朋友们知道以后，全部都觉得十分惊讶。

几个要好的朋友，晚上与我促膝长谈至夜深。他们不理解的无怪乎几个问题：

1. 你这几年的宝贵经验要作废了，不可惜么？
2. 你年纪这么大了，读研以后再工作，会更麻烦。
3. 你确定读研以后的生活会比现在更好么？
4. 你的社保怎么办？

最后的结果是：我要慎重考虑，谨慎选择。

其实对于同学的这些疑问，我也早已考虑过，如今的选择就是最后的决定了。

我也明白他们为何有如此顾虑：作为奔三的人来说，结婚生子才是“正事儿”，赚钱买房才是当务之急。这个时候，还将什么深造、梦想都是不智之举。

在生活压力下，每个人做出的选择都是对于自己最合理的。因此盲目地责备他们不敢跳出“舒适区”也是不负责任的。

当我辞职回家后，考研群里逐渐稀少的信息，无不反映出大家的激情已经日益消退。要知道这才刚刚一个月而已。

诸多的表现：

1. 三分钟热度
2. 害怕孤独，企图合群
3. 懒，拖延症
4. 容易上瘾

原因：

1. 失去了激情。年龄大，经历多以后，就对生活失去了激情。
2. 失去了自信。被生活重锤打击，承认了自己是失败者。
